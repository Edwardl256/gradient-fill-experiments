# gradient-fill-experiments
Experimenting with various ways of making a gradient fill that follows the contour of a shape. Some manual labour, and later using opencv.  
My goal here is trying to bring back an old gradient fill effect found in [Apex Media](https://www.leonik.net/dml/sec_apex.py). A video is included in here showing the results from that program.  
At the very least I would like to make the corners rounder.
